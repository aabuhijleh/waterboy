# 💧 Waterboy

## ⌨️ Usage

```bash
# install dependencies
$ npm install

# development mode
$ npm run dev

# production build
$ npm run build

# run the popup in the browser using DevServer
$ npm start
```

## 🛠️ Installation

- Open [`chrome://extensions`](chrome://extensions)
- Enable `Developer mode`
- Click `Load unpacked` and open the `dist` directory
